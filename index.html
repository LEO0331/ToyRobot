Toy Robot

Guide your toy robot with simple controls and see every move update live on the board.

How To Control

  • Start with PLACE X,Y,F before any move commands.
  • In Text mode, type single commands or run multiline script commands with Step/Run All.
  • In Button mode, select X, Y, and Facing then press PLACE, then use LEFT/MOVE/RIGHT/REPORT buttons.
  • Valid commands are uppercase only: PLACE, MOVE, LEFT, RIGHT, REPORT.
  • Grid range is 0..5. Any move that would fall is rejected and state remains unchanged.
